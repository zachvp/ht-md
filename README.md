# web-md

Pick any element on a webpage and copy it as Markdown.

## Build Environment

- macOS or Linux
- Node.js >= 18
- npm >= 9

## Build Instructions

1. Install dependencies:
   ```
   npm install
   ```

2. Build the extension:
   ```
   npm run build
   ```

This produces:
- `dist/` — the unpacked extension directory
- `web-md.xpi` — the packaged Firefox extension
