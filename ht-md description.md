# Usage & Behavior
## ☄️ Single Target
👆 Click extension icon
🎯 Hover to paint target
🌈 Click to capture target as Markdown on your clipboard

## 👾 Multi-Target 👾
Click extension icon
Hold ⌘ Cmd (Mac) or Ctrl (Windows/Linux)
Click on each target in the order you want to paint them
Press Enter/Return to confirm selection and copy to your clipboard
Use the left & right arrow keys to undo/redo target selections

# Configuration
Pretty much everything can be configured in the extension's config tab. Holler if something isn't that should be there.
